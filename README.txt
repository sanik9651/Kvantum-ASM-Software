Kvantum ASM

Программа для отображения телеметрии системы на OLED-дисплее через serial-подключение.

Что умеет:
- отображает выбранные показатели системы
- поддерживает выбор языка интерфейса
- поддерживает пресеты слотов
- поддерживает анимации рамки
- поддерживает автозапуск вместе с Windows
- сохраняет настройки в config.json

Требования:
- Windows 10 или Windows 11
- подключённое совместимое устройство по serial/COM
- драйверы и доступ к USB/COM-порту
- при необходимости: запуск от имени администратора

Первый запуск:
1. Подключите устройство к компьютеру, используя кабель USB Type C
2. Запустите KvantumASM.exe
3. Подождите, пока программа найдёт устройство
4. Выберите язык, пресет и нужные показатели
5. При необходимости включите автозапуск

Автозапуск:
- Переключатель AUTOSTART включает и выключает запуск программы при входе в систему
- Если автозапуск включён, Windows будет запускать программу автоматически при входе пользователя

Где сохраняются настройки:
- настройки сохраняются в папке:
  %LOCALAPPDATA%\KvantumASM

- основной файл настроек:
  config.json

Если программа не запускается или не видит устройство:
- проверьте, подключено ли устройство
- убедитесь, что COM-порт доступен
- попробуйте запустить программу от имени администратора (помогает в большинстве случаев)
- временно отключите антивирус, если он блокирует запуск

Примечание:
Некоторые антивирусы могут показывать предупреждение для новых .exe файлов. Это нормально для недавно собранных приложений без цифровой подписи.
